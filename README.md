Sales Analytics & Profitability (Excel)

## Project Overview

This project focuses on analyzing sales data using Microsoft Excel to generate meaningful business insights. The dataset contains 1500+ records including sales, profit, discount, and customer details.

## Objective

- Analyze sales performance across different regions and categories
- Understand the impact of discount on profitability
- Build an interactive dashboard for business decision-making

## Tools & Techniques Used

- Microsoft Excel
- Pivot Tables
- Charts
- Slicers
- Conditional Formatting

## Dataset Features

- Order ID, Order Date
- Customer Details
- Region & Category
- Sales, Quantity
- Discount
- Profit

## Dashboard Features

- Sales by Region
- Sales by Product
- Monthly Sales Trend
- Profit Analysis
- KPIs (Total Sales, Total Profit, Total Orders)
- Interactive Filters (Slicers)
- Highlighting high and low performance using Conditional Formatting

## Key Insights

- Higher discounts lead to lower profit margins
- Some transactions show negative profit due to heavy discounting
- Sales performance varies across regions
- Conditional formatting helps quickly identify loss-making areas
